# SPI Protocol Basics: A Comprehensive Guide

## Overview

The Serial Peripheral Interface (SPI) is a synchronous serial communication protocol widely used for short-distance communication between microcontrollers and peripheral devices. It's known for its simplicity, speed, and full-duplex capability.

## Architecture and Topology

### Master-Slave Configuration

SPI follows a master-slave architecture where one master device controls one or more slave devices. The master generates the clock signal and initiates all communication. Key characteristics include:

- **Single Master**: Only one device controls the bus at a time
- **Multiple Slaves**: The master can communicate with multiple slaves using separate chip select lines
- **Synchronous Communication**: Data transfer is synchronized by a clock signal generated by the master
- **Full-Duplex**: Simultaneous bidirectional data transfer

### Four-Wire Communication

SPI uses four main signal lines:

1. **SCLK (Serial Clock)**: Clock signal generated by the master
2. **MOSI (Master Out Slave In)**: Data line from master to slave
3. **MISO (Master In Slave Out)**: Data line from slave to master
4. **CS/SS (Chip Select/Slave Select)**: Slave selection line (active low, one per slave)

## Clock Polarity and Phase

SPI defines four modes based on Clock Polarity (CPOL) and Clock Phase (CPHA):

- **CPOL=0**: Clock idle state is low
- **CPOL=1**: Clock idle state is high
- **CPHA=0**: Data sampled on leading edge, shifted on trailing edge
- **CPHA=1**: Data shifted on leading edge, sampled on trailing edge

**SPI Modes:**
- Mode 0: CPOL=0, CPHA=0
- Mode 1: CPOL=0, CPHA=1
- Mode 2: CPOL=1, CPHA=0
- Mode 3: CPOL=1, CPHA=1

### C++ Implementation (Object-Oriented)

```cpp
#include <cstdint>
#include <vector>
#include <stdexcept>

class SPIMaster {
public:
    enum class Mode {
        MODE_0 = 0,  // CPOL=0, CPHA=0
        MODE_1 = 1,  // CPOL=0, CPHA=1
        MODE_2 = 2,  // CPOL=1, CPHA=0
        MODE_3 = 3   // CPOL=1, CPHA=1
    };
    
    enum class BitOrder {
        MSB_FIRST,
        LSB_FIRST
    };
    
    struct Config {
        uint32_t clock_speed;
        Mode mode;
        BitOrder bit_order;
        
        Config() : clock_speed(1000000), mode(Mode::MODE_0), 
                   bit_order(BitOrder::MSB_FIRST) {}
    };

private:
    Config config_;
    volatile uint32_t* spi_base_;
    bool initialized_;
    
    // Register offsets
    static constexpr uint32_t CR1_OFFSET = 0x00;
    static constexpr uint32_t CR2_OFFSET = 0x04;
    static constexpr uint32_t SR_OFFSET  = 0x08;
    static constexpr uint32_t DR_OFFSET  = 0x0C;
    
    // Helper methods for register access
    volatile uint32_t& CR1() { return *(spi_base_ + CR1_OFFSET / 4); }
    volatile uint32_t& SR()  { return *(spi_base_ + SR_OFFSET / 4); }
    volatile uint32_t& DR()  { return *(spi_base_ + DR_OFFSET / 4); }
    
    void waitTxEmpty() {
        while (!(SR() & (1 << 1)));  // Wait for TXE
    }
    
    void waitRxNotEmpty() {
        while (!(SR() & (1 << 0)));  // Wait for RXNE
    }

public:
    SPIMaster(uint32_t base_address) 
        : spi_base_(reinterpret_cast<volatile uint32_t*>(base_address)),
          initialized_(false) {}
    
    ~SPIMaster() {
        if (initialized_) {
            deinit();
        }
    }
    
    void init(const Config& config) {
        config_ = config;
        
        uint32_t cr1_value = (1 << 2);  // Master mode
        
        // Configure clock polarity and phase
        switch (config.mode) {
            case Mode::MODE_0:
                break;
            case Mode::MODE_1:
                cr1_value |= (1 << 0);  // CPHA
                break;
            case Mode::MODE_2:
                cr1_value |= (1 << 1);  // CPOL
                break;
            case Mode::MODE_3:
                cr1_value |= (1 << 0) | (1 << 1);  // CPHA | CPOL
                break;
        }
        
        // Configure bit order
        if (config.bit_order == BitOrder::LSB_FIRST) {
            cr1_value |= (1 << 7);
        }
        
        CR1() = cr1_value;
        CR1() |= (1 << 6);  // Enable SPI
        
        initialized_ = true;
    }
    
    void deinit() {
        CR1() &= ~(1 << 6);  // Disable SPI
        initialized_ = false;
    }
    
    uint8_t transfer(uint8_t data) {
        if (!initialized_) {
            throw std::runtime_error("SPI not initialized");
        }
        
        waitTxEmpty();
        DR() = data;
        waitRxNotEmpty();
        return static_cast<uint8_t>(DR());
    }
    
    std::vector<uint8_t> transfer(const std::vector<uint8_t>& tx_data) {
        std::vector<uint8_t> rx_data;
        rx_data.reserve(tx_data.size());
        
        for (uint8_t byte : tx_data) {
            rx_data.push_back(transfer(byte));
        }
        
        return rx_data;
    }
    
    void write(const std::vector<uint8_t>& data) {
        for (uint8_t byte : data) {
            transfer(byte);
        }
    }
    
    std::vector<uint8_t> read(size_t length) {
        std::vector<uint8_t> data;
        data.reserve(length);
        
        for (size_t i = 0; i < length; i++) {
            data.push_back(transfer(0xFF));  // Send dummy bytes
        }
        
        return data;
    }
};
```

```rust
use core::ptr::{read_volatile, write_volatile};

/// SPI Mode configuration
#[derive(Debug, Clone, Copy)]
pub enum SpiMode {
    Mode0,  // CPOL=0, CPHA=0
    Mode1,  // CPOL=0, CPHA=1
    Mode2,  // CPOL=1, CPHA=0
    Mode3,  // CPOL=1, CPHA=1
}

/// Bit order configuration
#[derive(Debug, Clone, Copy)]
pub enum BitOrder {
    MsbFirst,
    LsbFirst,
}

/// SPI Configuration
#[derive(Debug, Clone, Copy)]
pub struct SpiConfig {
    pub clock_speed: u32,
    pub mode: SpiMode,
    pub bit_order: BitOrder,
}

impl Default for SpiConfig {
    fn default() -> Self {
        Self {
            clock_speed: 1_000_000,  // 1 MHz
            mode: SpiMode::Mode0,
            bit_order: BitOrder::MsbFirst,
        }
    }
}

/// SPI Register structure
#[repr(C)]
struct SpiRegisters {
    cr1: u32,
    cr2: u32,
    sr: u32,
    dr: u32,
}

/// SPI Control Register 1 bits
mod cr1_bits {
    pub const CPHA: u32 = 1 << 0;
    pub const CPOL: u32 = 1 << 1;
    pub const MSTR: u32 = 1 << 2;
    pub const SPE: u32 = 1 << 6;
    pub const LSBFIRST: u32 = 1 << 7;
}

/// SPI Status Register bits
mod sr_bits {
    pub const RXNE: u32 = 1 << 0;  // Receive buffer not empty
    pub const TXE: u32 = 1 << 1;   // Transmit buffer empty
    pub const BSY: u32 = 1 << 7;   // Busy flag
}

/// SPI Master implementation
pub struct SpiMaster {
    registers: *mut SpiRegisters,
    config: SpiConfig,
}

impl SpiMaster {
    /// Create a new SPI master instance
    ///
    /// # Safety
    /// The base_address must point to valid SPI peripheral registers
    pub unsafe fn new(base_address: usize) -> Self {
        Self {
            registers: base_address as *mut SpiRegisters,
            config: SpiConfig::default(),
        }
    }
    
    /// Initialize the SPI peripheral with given configuration
    pub fn init(&mut self, config: SpiConfig) -> Result<(), &'static str> {
        self.config = config;
        
        let mut cr1_value = cr1_bits::MSTR;  // Master mode
        
        // Configure clock polarity and phase
        match config.mode {
            SpiMode::Mode0 => {},
            SpiMode::Mode1 => cr1_value |= cr1_bits::CPHA,
            SpiMode::Mode2 => cr1_value |= cr1_bits::CPOL,
            SpiMode::Mode3 => cr1_value |= cr1_bits::CPHA | cr1_bits::CPOL,
        }
        
        // Configure bit order
        if matches!(config.bit_order, BitOrder::LsbFirst) {
            cr1_value |= cr1_bits::LSBFIRST;
        }
        
        unsafe {
            // Write configuration
            write_volatile(&mut (*self.registers).cr1, cr1_value);
            
            // Enable SPI
            let cr1 = read_volatile(&(*self.registers).cr1);
            write_volatile(&mut (*self.registers).cr1, cr1 | cr1_bits::SPE);
        }
        
        Ok(())
    }
    
    /// Wait until transmit buffer is empty
    fn wait_tx_empty(&self) {
        unsafe {
            while (read_volatile(&(*self.registers).sr) & sr_bits::TXE) == 0 {}
        }
    }
    
    /// Wait until receive buffer is not empty
    fn wait_rx_not_empty(&self) {
        unsafe {
            while (read_volatile(&(*self.registers).sr) & sr_bits::RXNE) == 0 {}
        }
    }
    
    /// Transfer a single byte (full-duplex)
    pub fn transfer(&mut self, data: u8) -> u8 {
        self.wait_tx_empty();
        
        unsafe {
            write_volatile(&mut (*self.registers).dr, data as u32);
        }
        
        self.wait_rx_not_empty();
        
        unsafe {
            read_volatile(&(*self.registers).dr) as u8
        }
    }
    
    /// Transfer multiple bytes
    pub fn transfer_slice(&mut self, tx_buffer: &[u8], rx_buffer: &mut [u8]) 
        -> Result<(), &'static str> 
    {
        if tx_buffer.len() != rx_buffer.len() {
            return Err("Buffer length mismatch");
        }
        
        for (tx, rx) in tx_buffer.iter().zip(rx_buffer.iter_mut()) {
            *rx = self.transfer(*tx);
        }
        
        Ok(())
    }
    
    /// Write data (ignore received data)
    pub fn write(&mut self, data: &[u8]) {
        for &byte in data {
            self.transfer(byte);
        }
    }
    
    /// Read data (send dummy bytes)
    pub fn read(&mut self, buffer: &mut [u8]) {
        for byte in buffer.iter_mut() {
            *byte = self.transfer(0xFF);
        }
    }
}
```

## Key Takeaways

The SPI protocol offers several advantages for embedded systems:
- High-speed communication (often 10+ MHz)
- Full-duplex operation for simultaneous send/receive
- Simple hardware implementation with no need for start/stop bits
- Flexible clock polarity and phase configuration
- No acknowledgment mechanism keeps protocol simple

The main limitation is the requirement for a separate chip select line for each slave device, which can consume many GPIO pins in systems with multiple peripherals.
