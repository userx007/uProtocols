# SPI (Serial Peripheral Interface) Programming Guide

## Overview

SPI is a synchronous serial communication protocol used for short-distance communication between microcontrollers and peripheral devices like sensors, SD cards, displays, and ADCs. It operates in a master-slave architecture with full-duplex communication.

## Key Characteristics

**Bus Signals:**
- **MOSI** (Master Out Slave In): Data from master to slave
- **MISO** (Master In Slave Out): Data from slave to master
- **SCLK** (Serial Clock): Clock signal generated by master
- **SS/CS** (Slave Select/Chip Select): Device selection line

**Features:**
- Full-duplex synchronous communication
- High-speed data transfer (MHz range)
- Simple hardware interface
- Multiple slaves on same bus (separate CS lines)
- No standard protocol - implementation varies by device

## Clock Gating in SPI

Clock gating is a power optimization technique that disables the SPI clock when the peripheral is idle, reducing dynamic power consumption. This is especially important in battery-powered embedded systems.

### Benefits:
- Reduces dynamic power consumption
- Extends battery life in portable devices
- Minimizes electromagnetic interference (EMI)
- Decreases heat generation

### Implementation Approaches:
1. **Hardware clock gating**: Built into MCU peripheral design
2. **Software clock gating**: Manual enable/disable of peripheral clocks
3. **Automatic gating**: MCU automatically gates clocks during WFI/sleep modes

## C/C++ Implementation Examples

### Basic SPI Driver with Clock Gating (STM32-style)

```c
#include <stdint.h>
#include <stdbool.h>

// Register definitions (example for ARM Cortex-M)
#define RCC_APB2ENR    (*(volatile uint32_t*)0x40021018)
#define SPI1_CR1       (*(volatile uint32_t*)0x40013000)
#define SPI1_DR        (*(volatile uint32_t*)0x4001300C)
#define SPI1_SR        (*(volatile uint32_t*)0x40013008)

#define RCC_SPI1_EN    (1 << 12)
#define SPI_CR1_SPE    (1 << 6)   // SPI Enable
#define SPI_SR_TXE     (1 << 1)   // Transmit buffer empty
#define SPI_SR_RXNE    (1 << 0)   // Receive buffer not empty
#define SPI_SR_BSY     (1 << 7)   // Busy flag

typedef struct {
    bool clock_enabled;
    uint32_t transfer_count;
} spi_state_t;

static spi_state_t spi_state = {false, 0};

// Enable SPI clock
void spi_enable_clock(void) {
    if (!spi_state.clock_enabled) {
        RCC_APB2ENR |= RCC_SPI1_EN;
        spi_state.clock_enabled = true;
    }
}

// Disable SPI clock (clock gating)
void spi_disable_clock(void) {
    if (spi_state.clock_enabled) {
        // Wait until SPI is not busy
        while (SPI1_SR & SPI_SR_BSY);
        
        // Disable SPI peripheral
        SPI1_CR1 &= ~SPI_CR1_SPE;
        
        // Gate the clock
        RCC_APB2ENR &= ~RCC_SPI1_EN;
        spi_state.clock_enabled = false;
    }
}

// Initialize SPI with clock gating support
void spi_init(void) {
    spi_enable_clock();
    
    // Configure SPI (Master mode, 8-bit, MSB first, etc.)
    SPI1_CR1 = (1 << 2) |  // Master mode
               (3 << 3) |  // Baud rate = fPCLK/16
               (1 << 9) |  // Software slave management
               (1 << 8);   // Internal slave select
    
    // Enable SPI
    SPI1_CR1 |= SPI_CR1_SPE;
}

// Transfer single byte with auto clock gating
uint8_t spi_transfer_byte(uint8_t data) {
    // Enable clock if needed
    spi_enable_clock();
    
    // Enable SPI if disabled
    if (!(SPI1_CR1 & SPI_CR1_SPE)) {
        SPI1_CR1 |= SPI_CR1_SPE;
    }
    
    // Wait for TX buffer empty
    while (!(SPI1_SR & SPI_SR_TXE));
    
    // Send data
    SPI1_DR = data;
    
    // Wait for RX buffer not empty
    while (!(SPI1_SR & SPI_SR_RXNE));
    
    // Read received data
    uint8_t received = SPI1_DR;
    
    spi_state.transfer_count++;
    
    return received;
}

// Transfer multiple bytes with optimized clock gating
void spi_transfer_buffer(const uint8_t* tx_data, uint8_t* rx_data, 
                         uint16_t length) {
    if (length == 0) return;
    
    // Enable clock once for entire transfer
    spi_enable_clock();
    
    if (!(SPI1_CR1 & SPI_CR1_SPE)) {
        SPI1_CR1 |= SPI_CR1_SPE;
    }
    
    for (uint16_t i = 0; i < length; i++) {
        while (!(SPI1_SR & SPI_SR_TXE));
        SPI1_DR = tx_data ? tx_data[i] : 0xFF;
        
        while (!(SPI1_SR & SPI_SR_RXNE));
        uint8_t received = SPI1_DR;
        
        if (rx_data) {
            rx_data[i] = received;
        }
    }
    
    // Wait for final transaction to complete
    while (SPI1_SR & SPI_SR_BSY);
    
    spi_state.transfer_count += length;
}

// Enter low power mode (aggressive clock gating)
void spi_enter_low_power(void) {
    spi_disable_clock();
}

// Example usage with power-aware pattern
void read_sensor_with_clock_gating(void) {
    uint8_t sensor_data[10];
    
    // Clock is enabled automatically during transfer
    spi_transfer_buffer(NULL, sensor_data, 10);
    
    // Explicitly gate clock when done
    spi_disable_clock();
    
    // Process sensor data while SPI clock is gated
    // ... data processing ...
}
```

### C++ RAII-based SPI with Automatic Clock Gating

```cpp
#include <cstdint>
#include <array>
#include <optional>

class SpiClockGuard {
public:
    SpiClockGuard() {
        // Enable SPI clock
        RCC_APB2ENR |= RCC_SPI1_EN;
        
        // Enable SPI peripheral
        if (!(SPI1_CR1 & SPI_CR1_SPE)) {
            SPI1_CR1 |= SPI_CR1_SPE;
        }
    }
    
    ~SpiClockGuard() {
        // Wait until not busy
        while (SPI1_SR & SPI_SR_BSY);
        
        // Disable SPI
        SPI1_CR1 &= ~SPI_CR1_SPE;
        
        // Gate clock
        RCC_APB2ENR &= ~RCC_SPI1_EN;
    }
    
    // Non-copyable, non-movable
    SpiClockGuard(const SpiClockGuard&) = delete;
    SpiClockGuard& operator=(const SpiClockGuard&) = delete;
};

class SpiMaster {
private:
    bool initialized_{false};
    
    uint8_t transfer_byte_internal(uint8_t data) {
        while (!(SPI1_SR & SPI_SR_TXE));
        SPI1_DR = data;
        
        while (!(SPI1_SR & SPI_SR_RXNE));
        return static_cast<uint8_t>(SPI1_DR);
    }
    
public:
    void init() {
        // Initial setup (done once)
        SpiClockGuard guard;
        
        SPI1_CR1 = (1 << 2) | (3 << 3) | (1 << 9) | (1 << 8);
        SPI1_CR1 |= SPI_CR1_SPE;
        
        initialized_ = true;
    }
    
    // Single byte transfer with RAII clock management
    uint8_t transfer(uint8_t data) {
        SpiClockGuard guard;  // Clock enabled here
        return transfer_byte_internal(data);
    }  // Clock automatically gated when guard goes out of scope
    
    // Buffer transfer with RAII clock management
    template<size_t N>
    std::array<uint8_t, N> transfer(const std::array<uint8_t, N>& tx_data) {
        SpiClockGuard guard;  // Single clock enable for entire transfer
        std::array<uint8_t, N> rx_data;
        
        for (size_t i = 0; i < N; ++i) {
            rx_data[i] = transfer_byte_internal(tx_data[i]);
        }
        
        return rx_data;
    }  // Clock automatically gated here
    
    // Read-only operation
    template<size_t N>
    std::array<uint8_t, N> read() {
        SpiClockGuard guard;
        std::array<uint8_t, N> data;
        
        for (auto& byte : data) {
            byte = transfer_byte_internal(0xFF);
        }
        
        return data;
    }
};

// Usage example
void example_usage() {
    SpiMaster spi;
    spi.init();
    
    // Clock is automatically managed
    auto response = spi.transfer<4>({0x01, 0x02, 0x03, 0x04});
    // Clock is now gated
    
    // Process data while clock is off
    for (auto byte : response) {
        // ... process ...
    }
    
    // Another transaction - clock enabled again
    auto sensor_data = spi.read<8>();
    // Clock gated again
}
```

## Rust Implementation

```rust
use core::ptr::{read_volatile, write_volatile};

// Memory-mapped register addresses
const RCC_APB2ENR: *mut u32 = 0x40021018 as *mut u32;
const SPI1_CR1: *mut u32 = 0x40013000 as *mut u32;
const SPI1_DR: *mut u32 = 0x4001300C as *mut u32;
const SPI1_SR: *mut u32 = 0x40013008 as *mut u32;

// Bit definitions
const RCC_SPI1_EN: u32 = 1 << 12;
const SPI_CR1_SPE: u32 = 1 << 6;
const SPI_SR_TXE: u32 = 1 << 1;
const SPI_SR_RXNE: u32 = 1 << 0;
const SPI_SR_BSY: u32 = 1 << 7;

/// RAII guard for SPI clock management
pub struct SpiClockGuard;

impl SpiClockGuard {
    pub fn new() -> Self {
        unsafe {
            // Enable SPI clock
            let rcc = read_volatile(RCC_APB2ENR);
            write_volatile(RCC_APB2ENR, rcc | RCC_SPI1_EN);
            
            // Enable SPI peripheral
            let cr1 = read_volatile(SPI1_CR1);
            if (cr1 & SPI_CR1_SPE) == 0 {
                write_volatile(SPI1_CR1, cr1 | SPI_CR1_SPE);
            }
        }
        SpiClockGuard
    }
}

impl Drop for SpiClockGuard {
    fn drop(&mut self) {
        unsafe {
            // Wait until not busy
            while (read_volatile(SPI1_SR) & SPI_SR_BSY) != 0 {}
            
            // Disable SPI
            let cr1 = read_volatile(SPI1_CR1);
            write_volatile(SPI1_CR1, cr1 & !SPI_CR1_SPE);
            
            // Gate clock
            let rcc = read_volatile(RCC_APB2ENR);
            write_volatile(RCC_APB2ENR, rcc & !RCC_SPI1_EN);
        }
    }
}

pub struct SpiMaster {
    initialized: bool,
}

impl SpiMaster {
    pub const fn new() -> Self {
        SpiMaster { initialized: false }
    }
    
    pub fn init(&mut self) {
        let _guard = SpiClockGuard::new();
        
        unsafe {
            // Configure SPI: Master mode, /16 prescaler, software SS
            write_volatile(SPI1_CR1, 
                (1 << 2) |  // Master mode
                (3 << 3) |  // Baud rate /16
                (1 << 9) |  // Software slave management
                (1 << 8)    // Internal slave select
            );
            
            // Enable SPI
            let cr1 = read_volatile(SPI1_CR1);
            write_volatile(SPI1_CR1, cr1 | SPI_CR1_SPE);
        }
        
        self.initialized = true;
    }
    
    /// Transfer single byte with automatic clock gating
    pub fn transfer(&self, data: u8) -> u8 {
        let _guard = SpiClockGuard::new();
        self.transfer_byte_internal(data)
    }
    
    fn transfer_byte_internal(&self, data: u8) -> u8 {
        unsafe {
            // Wait for TX buffer empty
            while (read_volatile(SPI1_SR) & SPI_SR_TXE) == 0 {}
            
            // Send data
            write_volatile(SPI1_DR, data as u32);
            
            // Wait for RX buffer not empty
            while (read_volatile(SPI1_SR) & SPI_SR_RXNE) == 0 {}
            
            // Read received data
            read_volatile(SPI1_DR) as u8
        }
    }
    
    /// Transfer buffer with automatic clock gating
    pub fn transfer_buffer(&self, tx_data: &[u8], rx_data: &mut [u8]) {
        assert_eq!(tx_data.len(), rx_data.len());
        
        // Single clock guard for entire transfer
        let _guard = SpiClockGuard::new();
        
        for (tx, rx) in tx_data.iter().zip(rx_data.iter_mut()) {
            *rx = self.transfer_byte_internal(*tx);
        }
    }
    
    /// Read multiple bytes
    pub fn read_buffer(&self, buffer: &mut [u8]) {
        let _guard = SpiClockGuard::new();
        
        for byte in buffer.iter_mut() {
            *byte = self.transfer_byte_internal(0xFF);
        }
    }
    
    /// Write multiple bytes (ignore response)
    pub fn write_buffer(&self, buffer: &[u8]) {
        let _guard = SpiClockGuard::new();
        
        for &byte in buffer {
            self.transfer_byte_internal(byte);
        }
    }
}

// Example: SPI device driver with clock gating
pub struct Mcp3008Adc {
    spi: SpiMaster,
}

impl Mcp3008Adc {
    pub fn new(mut spi: SpiMaster) -> Self {
        spi.init();
        Mcp3008Adc { spi }
    }
    
    /// Read ADC channel with automatic clock management
    pub fn read_channel(&self, channel: u8) -> u16 {
        assert!(channel < 8);
        
        // Clock is automatically enabled by transfer()
        let cmd = [
            0x01,
            (0x80 | (channel << 4)),
            0x00,
        ];
        
        let response = [
            self.spi.transfer(cmd[0]),
            self.spi.transfer(cmd[1]),
            self.spi.transfer(cmd[2]),
        ];
        // Clock is now gated
        
        // Parse 10-bit result
        let value = (((response[1] as u16) & 0x03) << 8) | (response[2] as u16);
        value
    }
}

// Usage example
pub fn example_usage() {
    let spi = SpiMaster::new();
    let adc = Mcp3008Adc::new(spi);
    
    // Read from channel 0 - clock automatically managed
    let reading = adc.read_channel(0);
    
    // Clock is gated here - process data
    let voltage = (reading as f32 / 1023.0) * 3.3;
    
    // Next read will re-enable clock
    let reading2 = adc.read_channel(1);
}
```

## Summary

**SPI Clock Gating** is a critical power optimization technique that disables the SPI peripheral clock when not actively transferring data. The implementations shown demonstrate:

**Key Concepts:**
- Manual clock enable/disable in C reduces power consumption between transfers
- RAII patterns in C++ and Rust provide automatic, safe clock management
- Single clock enable for buffer transfers is more efficient than per-byte gating
- Clock gating pairs well with low-power modes in embedded systems

**Best Practices:**
1. Enable clock only during active transfers
2. Use RAII guards to prevent forgetting to gate clocks
3. Batch transfers to minimize enable/disable overhead
4. Verify SPI is idle before disabling clocks
5. Consider automatic hardware gating in modern MCUs

**Power Savings:**
- Can reduce SPI subsystem power by 60-90% during idle periods
- Essential for battery-powered IoT and wearable devices
- Minimal performance impact when properly implemented

The code examples show progressive sophistication from basic C to modern Rust, all implementing the same core principle: only consume power when actually communicating over SPI.
