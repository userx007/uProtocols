## Introduction

SPI (Serial Peripheral Interface) is a synchronous, full-duplex, master-slave communication protocol widely used in embedded systems to connect microcontrollers and SoCs to peripherals such as flash memory, sensors, display controllers, ADCs, and DACs.

In Linux, SPI devices are managed through a layered subsystem. At the lowest layer sits the **SPI Master Controller Driver** — a platform-specific kernel driver responsible for:

- Initializing and configuring the SPI hardware block on a given SoC or platform.
- Managing clock generation, chip select (CS) lines, and bus arbitration.
- Transferring data to/from SPI slave devices via DMA or PIO (programmed I/O).
- Exposing a standard `spi_master` (or `spi_controller` in modern kernels) interface to upper layers.

This document covers the architecture, key data structures, registration patterns, and practical code examples in both **C/C++** and **Rust** for implementing such a driver.

## SPI Protocol Fundamentals

### Signal Lines

| Signal | Direction | Description |
|--------|-----------|-------------|
| SCLK   | Master → Slave | Serial clock, generated by the master |
| MOSI   | Master → Slave | Master Out Slave In — data from master |
| MISO   | Slave → Master | Master In Slave Out — data from slave |
| CS/SS  | Master → Slave | Chip Select / Slave Select (active low) |

### Clock Modes (CPOL/CPHA)

| Mode | CPOL | CPHA | Clock Idle | Data Captured |
|------|------|------|------------|---------------|
| 0    | 0    | 0    | Low        | Rising edge   |
| 1    | 0    | 1    | Low        | Falling edge  |
| 2    | 1    | 0    | High       | Falling edge  |
| 3    | 1    | 1    | High       | Rising edge   |

### Transfer Characteristics

- Full-duplex: MOSI and MISO transfer simultaneously.
- No protocol overhead: pure shift register mechanism.
- Bit order: typically MSB first, but LSB-first mode is supported by many controllers.
- Word size: typically 8-bit, but 4–32 bit words are common in hardware.

## SPI Master Controller Driver Structure

### Key Kernel Data Structures

```c
/* The central structure representing a SPI master controller */
struct spi_controller {           // Aliased as spi_master in older kernels
    struct device   dev;
    int             bus_num;
    u16             num_chipselect;
    u32             min_speed_hz;
    u32             max_speed_hz;

    /* Callbacks implemented by the controller driver */
    int  (*setup)(struct spi_device *spi);
    int  (*transfer_one)(struct spi_controller *ctlr,
                         struct spi_device *spi,
                         struct spi_transfer *transfer);
    int  (*transfer_one_message)(struct spi_controller *ctlr,
                                 struct spi_message *mesg);
    void (*set_cs)(struct spi_device *spi, bool enable);
    int  (*prepare_transfer_hardware)(struct spi_controller *ctlr);
    int  (*unprepare_transfer_hardware)(struct spi_controller *ctlr);
    /* ... more fields ... */
};

/* Represents a single SPI transfer segment */
struct spi_transfer {
    const void  *tx_buf;     /* Data to send (NULL = send zeros) */
    void        *rx_buf;     /* Buffer to receive data (NULL = discard) */
    unsigned    len;         /* Length in bytes */
    u32         speed_hz;    /* Transfer clock (0 = use spi_device default) */
    u8          bits_per_word;
    u16         delay_usecs; /* Delay after this transfer */
    dma_addr_t  tx_dma;     /* DMA address for tx_buf */
    dma_addr_t  rx_dma;     /* DMA address for rx_buf */
    /* ... */
};

/* A complete SPI message (list of transfers) */
struct spi_message {
    struct list_head    transfers;   /* Linked list of spi_transfer */
    struct spi_device  *spi;
    void (*complete)(void *context); /* Completion callback */
    void               *context;
    int                 status;      /* 0 on success */
};
```

## Implementing a Platform SPI Controller Driver in C

Below is a complete, annotated example of a minimal but realistic SPI master controller driver for a fictional `mysoc` platform.

### 1. Header and Register Definitions

```c
// drivers/spi/spi-mysoc.c

#include <linux/clk.h>
#include <linux/completion.h>
#include <linux/delay.h>
#include <linux/dma-mapping.h>
#include <linux/interrupt.h>
#include <linux/io.h>
#include <linux/kernel.h>
#include <linux/module.h>
#include <linux/of.h>
#include <linux/platform_device.h>
#include <linux/pm_runtime.h>
#include <linux/spi/spi.h>

#define DRIVER_NAME     "spi-mysoc"

/* Hardware register offsets */
#define MYSOC_SPI_CTRL      0x00   /* Control register */
#define MYSOC_SPI_STATUS    0x04   /* Status register */
#define MYSOC_SPI_CLK_DIV   0x08   /* Clock divider */
#define MYSOC_SPI_TX_DATA   0x0C   /* TX FIFO */
#define MYSOC_SPI_RX_DATA   0x10   /* RX FIFO */
#define MYSOC_SPI_CS        0x14   /* Chip select control */
#define MYSOC_SPI_INT_EN    0x18   /* Interrupt enable */
#define MYSOC_SPI_INT_STAT  0x1C   /* Interrupt status */

/* Control register bits */
#define CTRL_ENABLE         BIT(0)
#define CTRL_MASTER         BIT(1)
#define CTRL_CPOL           BIT(2)
#define CTRL_CPHA           BIT(3)
#define CTRL_LSB_FIRST      BIT(4)
#define CTRL_WORD_SIZE_SHIFT 8      /* bits [11:8] = word size - 1 */
#define CTRL_WORD_SIZE_MASK  (0xF << CTRL_WORD_SIZE_SHIFT)

/* Status register bits */
#define STATUS_TX_FULL      BIT(0)
#define STATUS_TX_EMPTY     BIT(1)
#define STATUS_RX_FULL      BIT(2)
#define STATUS_RX_EMPTY     BIT(3)
#define STATUS_BUSY         BIT(4)

/* Interrupt bits */
#define INT_TX_EMPTY        BIT(0)
#define INT_RX_FULL         BIT(1)
#define INT_TRANSFER_DONE   BIT(2)
#define INT_ERROR           BIT(3)

#define MYSOC_SPI_FIFO_DEPTH    16
#define MYSOC_SPI_MAX_CS        4
```

### 2. Driver Private Data Structure

```c
/**
 * struct mysoc_spi - Private driver data
 * @base:       Virtual address of the controller's MMIO region
 * @clk:        Reference to the SPI functional clock
 * @irq:        IRQ number
 * @ctlr:       Pointer back to spi_controller
 * @completion: Used to wait for interrupt-driven transfers
 * @tx_buf:     Current TX pointer during an active transfer
 * @rx_buf:     Current RX pointer during an active transfer
 * @tx_len:     Remaining TX bytes
 * @rx_len:     Remaining RX bytes
 * @lock:       Protects concurrent access to driver state
 */
struct mysoc_spi {
    void __iomem        *base;
    struct clk          *clk;
    int                  irq;
    struct spi_controller *ctlr;
    struct completion    completion;

    /* Active transfer state */
    const u8            *tx_buf;
    u8                  *rx_buf;
    int                  tx_len;
    int                  rx_len;
    int                  error;

    spinlock_t           lock;
};

/* Convenience MMIO accessors */
static inline u32 mysoc_rd(struct mysoc_spi *priv, u32 reg)
{
    return readl(priv->base + reg);
}

static inline void mysoc_wr(struct mysoc_spi *priv, u32 reg, u32 val)
{
    writel(val, priv->base + reg);
}

static inline void mysoc_set_bits(struct mysoc_spi *priv, u32 reg, u32 bits)
{
    mysoc_wr(priv, reg, mysoc_rd(priv, reg) | bits);
}

static inline void mysoc_clr_bits(struct mysoc_spi *priv, u32 reg, u32 bits)
{
    mysoc_wr(priv, reg, mysoc_rd(priv, reg) & ~bits);
}
```

### 3. Hardware Initialization

```c
/**
 * mysoc_hw_init - Initialize the SPI hardware block
 */
static int mysoc_hw_init(struct mysoc_spi *priv)
{
    u32 ctrl;

    /* Disable controller during setup */
    mysoc_wr(priv, MYSOC_SPI_CTRL, 0);

    /* Set master mode, enable controller */
    ctrl = CTRL_ENABLE | CTRL_MASTER;
    /* Default: 8-bit words, CPOL=0, CPHA=0, MSB first */
    ctrl |= ((8 - 1) << CTRL_WORD_SIZE_SHIFT);
    mysoc_wr(priv, MYSOC_SPI_CTRL, ctrl);

    /* Enable transfer-done and error interrupts */
    mysoc_wr(priv, MYSOC_SPI_INT_EN, INT_TRANSFER_DONE | INT_ERROR);

    return 0;
}

/**
 * mysoc_set_clock - Configure the SPI clock divider
 * @priv:       Driver private data
 * @speed_hz:   Requested SPI clock frequency
 *
 * Returns: actual configured frequency
 */
static u32 mysoc_set_clock(struct mysoc_spi *priv, u32 speed_hz)
{
    unsigned long clk_hz = clk_get_rate(priv->clk);
    u32 divider;

    if (speed_hz == 0 || clk_hz == 0)
        return 0;

    /* Divider: clk_hz / speed_hz, rounded up, minimum 2 */
    divider = DIV_ROUND_UP(clk_hz, speed_hz);
    if (divider < 2)
        divider = 2;
    /* Divider must be even for 50% duty cycle */
    divider = ALIGN(divider, 2);

    mysoc_wr(priv, MYSOC_SPI_CLK_DIV, divider);
    return clk_hz / divider;
}
```

### 4. The `setup` Callback

```c
/**
 * mysoc_spi_setup - Called when a new spi_device is registered or reconfigured.
 * Validate and apply device-specific settings.
 */
static int mysoc_spi_setup(struct spi_device *spi)
{
    struct mysoc_spi *priv = spi_controller_get_devdata(spi->controller);
    u32 ctrl;

    if (spi->bits_per_word < 4 || spi->bits_per_word > 16) {
        dev_err(&spi->dev, "Unsupported word size: %d bits\n",
                spi->bits_per_word);
        return -EINVAL;
    }

    if (spi->max_speed_hz > 50000000) {
        dev_err(&spi->dev, "Speed %u Hz exceeds maximum 50 MHz\n",
                spi->max_speed_hz);
        return -EINVAL;
    }

    /* Apply CPOL/CPHA mode */
    ctrl = mysoc_rd(priv, MYSOC_SPI_CTRL);
    ctrl &= ~(CTRL_CPOL | CTRL_CPHA);
    if (spi->mode & SPI_CPOL)
        ctrl |= CTRL_CPOL;
    if (spi->mode & SPI_CPHA)
        ctrl |= CTRL_CPHA;
    if (spi->mode & SPI_LSB_FIRST)
        ctrl |= CTRL_LSB_FIRST;

    /* Set word size */
    ctrl &= ~CTRL_WORD_SIZE_MASK;
    ctrl |= ((spi->bits_per_word - 1) << CTRL_WORD_SIZE_SHIFT);
    mysoc_wr(priv, MYSOC_SPI_CTRL, ctrl);

    dev_dbg(&spi->dev, "Setup: %d bpw, mode %d, %u Hz\n",
            spi->bits_per_word, spi->mode & 0x3, spi->max_speed_hz);
    return 0;
}
```

### 5. Chip Select Control

```c
/**
 * mysoc_set_cs - Assert or deassert a chip select line
 * @spi:    The SPI device
 * @enable: true = assert CS (select device), false = deassert
 *
 * Note: The SPI core manages CS polarity. By the time this is called,
 * `enable` already accounts for CS_HIGH flag.
 */
static void mysoc_set_cs(struct spi_device *spi, bool enable)
{
    struct mysoc_spi *priv = spi_controller_get_devdata(spi->controller);
    u32 cs_reg = mysoc_rd(priv, MYSOC_SPI_CS);

    if (enable)
        cs_reg |=  BIT(spi_get_chipselect(spi, 0));
    else
        cs_reg &= ~BIT(spi_get_chipselect(spi, 0));

    mysoc_wr(priv, MYSOC_SPI_CS, cs_reg);
}
```

### 6. FIFO Fill/Drain Helpers

```c
/**
 * mysoc_fill_tx_fifo - Push data into the hardware TX FIFO
 */
static void mysoc_fill_tx_fifo(struct mysoc_spi *priv)
{
    u32 status;
    int count = 0;

    while (priv->tx_len > 0 && count < MYSOC_SPI_FIFO_DEPTH) {
        status = mysoc_rd(priv, MYSOC_SPI_STATUS);
        if (status & STATUS_TX_FULL)
            break;

        if (priv->tx_buf)
            mysoc_wr(priv, MYSOC_SPI_TX_DATA, *priv->tx_buf++);
        else
            mysoc_wr(priv, MYSOC_SPI_TX_DATA, 0x00); /* dummy byte */

        priv->tx_len--;
        count++;
    }
}

/**
 * mysoc_drain_rx_fifo - Read data from the hardware RX FIFO
 */
static void mysoc_drain_rx_fifo(struct mysoc_spi *priv)
{
    u32 status, data;

    while (priv->rx_len > 0) {
        status = mysoc_rd(priv, MYSOC_SPI_STATUS);
        if (status & STATUS_RX_EMPTY)
            break;

        data = mysoc_rd(priv, MYSOC_SPI_RX_DATA);
        if (priv->rx_buf)
            *priv->rx_buf++ = (u8)data;
        priv->rx_len--;
    }
}
```

### 7. Interrupt Handler

```c
/**
 * mysoc_spi_irq - Interrupt service routine
 *
 * The controller fires an interrupt when:
 *   - The transfer is complete (INT_TRANSFER_DONE)
 *   - An error occurred (INT_ERROR)
 */
static irqreturn_t mysoc_spi_irq(int irq, void *dev_id)
{
    struct mysoc_spi *priv = dev_id;
    u32 stat;

    stat = mysoc_rd(priv, MYSOC_SPI_INT_STAT);
    /* Acknowledge all pending interrupts */
    mysoc_wr(priv, MYSOC_SPI_INT_STAT, stat);

    if (stat & INT_ERROR) {
        dev_err_ratelimited(&priv->ctlr->dev, "SPI transfer error\n");
        priv->error = -EIO;
        complete(&priv->completion);
        return IRQ_HANDLED;
    }

    if (stat & INT_TRANSFER_DONE) {
        /* Drain any remaining RX data */
        mysoc_drain_rx_fifo(priv);

        if (priv->tx_len > 0) {
            /* More data to send; refill TX FIFO */
            mysoc_fill_tx_fifo(priv);
        } else {
            /* Transfer fully complete */
            complete(&priv->completion);
        }
    }

    return IRQ_HANDLED;
}
```

### 8. The Core `transfer_one` Callback

```c
/**
 * mysoc_transfer_one - Execute a single spi_transfer
 *
 * Called by the SPI core for each transfer in a message.
 * Returns 0 if the transfer is complete synchronously,
 * or 1 if it will complete asynchronously (via spi_finalize_current_transfer).
 */
static int mysoc_transfer_one(struct spi_controller *ctlr,
                               struct spi_device *spi,
                               struct spi_transfer *xfer)
{
    struct mysoc_spi *priv = spi_controller_get_devdata(ctlr);
    unsigned long timeout;

    /* Configure clock for this transfer */
    mysoc_set_clock(priv, xfer->speed_hz ?: spi->max_speed_hz);

    /* Initialise transfer state */
    priv->tx_buf  = xfer->tx_buf;
    priv->rx_buf  = xfer->rx_buf;
    priv->tx_len  = xfer->len;
    priv->rx_len  = xfer->len;
    priv->error   = 0;
    reinit_completion(&priv->completion);

    /* Kick off the transfer by filling the TX FIFO */
    mysoc_fill_tx_fifo(priv);

    /* Wait for interrupt-driven completion (1 second timeout) */
    timeout = wait_for_completion_timeout(&priv->completion,
                                          msecs_to_jiffies(1000));
    if (!timeout) {
        dev_err(&ctlr->dev, "SPI transfer timed out\n");
        return -ETIMEDOUT;
    }

    return priv->error;
}
```

### 9. Probe and Remove

```c
/**
 * mysoc_spi_probe - Platform driver probe function
 *
 * Allocates resources, initializes hardware, registers the controller.
 */
static int mysoc_spi_probe(struct platform_device *pdev)
{
    struct spi_controller *ctlr;
    struct mysoc_spi *priv;
    struct resource *res;
    int ret;

    /* Allocate spi_controller with space for our private data */
    ctlr = devm_spi_alloc_host(&pdev->dev, sizeof(*priv));
    if (!ctlr)
        return -ENOMEM;

    priv = spi_controller_get_devdata(ctlr);
    priv->ctlr = ctlr;
    spin_lock_init(&priv->lock);
    init_completion(&priv->completion);

    /* Map MMIO registers */
    priv->base = devm_platform_ioremap_resource(pdev, 0);
    if (IS_ERR(priv->base))
        return PTR_ERR(priv->base);

    /* Get and enable the functional clock */
    priv->clk = devm_clk_get(&pdev->dev, "spi_clk");
    if (IS_ERR(priv->clk)) {
        dev_err(&pdev->dev, "Failed to get SPI clock\n");
        return PTR_ERR(priv->clk);
    }

    ret = clk_prepare_enable(priv->clk);
    if (ret) {
        dev_err(&pdev->dev, "Failed to enable SPI clock\n");
        return ret;
    }

    /* Request IRQ */
    priv->irq = platform_get_irq(pdev, 0);
    if (priv->irq < 0)
        return priv->irq;

    ret = devm_request_irq(&pdev->dev, priv->irq, mysoc_spi_irq,
                           0, DRIVER_NAME, priv);
    if (ret) {
        dev_err(&pdev->dev, "Failed to request IRQ %d\n", priv->irq);
        return ret;
    }

    /* Initialize the hardware */
    ret = mysoc_hw_init(priv);
    if (ret) {
        dev_err(&pdev->dev, "Hardware initialization failed\n");
        return ret;
    }

    /* Populate spi_controller fields */
    platform_set_drvdata(pdev, ctlr);
    ctlr->dev.of_node     = pdev->dev.of_node;
    ctlr->bus_num         = -1;          /* Auto-assign bus number */
    ctlr->num_chipselect  = MYSOC_SPI_MAX_CS;
    ctlr->mode_bits       = SPI_CPOL | SPI_CPHA | SPI_LSB_FIRST | SPI_CS_HIGH;
    ctlr->bits_per_word_mask = SPI_BPW_RANGE_MASK(4, 16);
    ctlr->min_speed_hz    = clk_get_rate(priv->clk) / 65534;
    ctlr->max_speed_hz    = 50000000; /* 50 MHz */
    ctlr->setup           = mysoc_spi_setup;
    ctlr->set_cs          = mysoc_set_cs;
    ctlr->transfer_one    = mysoc_transfer_one;

    /* Register the controller with the SPI core */
    ret = devm_spi_register_controller(&pdev->dev, ctlr);
    if (ret) {
        dev_err(&pdev->dev, "Failed to register SPI controller\n");
        clk_disable_unprepare(priv->clk);
        return ret;
    }

    dev_info(&pdev->dev, "MySoC SPI master registered (bus %d, IRQ %d)\n",
             ctlr->bus_num, priv->irq);
    return 0;
}

static int mysoc_spi_remove(struct platform_device *pdev)
{
    struct spi_controller *ctlr = platform_get_drvdata(pdev);
    struct mysoc_spi *priv = spi_controller_get_devdata(ctlr);

    /* Disable hardware */
    mysoc_wr(priv, MYSOC_SPI_CTRL, 0);
    mysoc_wr(priv, MYSOC_SPI_INT_EN, 0);
    clk_disable_unprepare(priv->clk);

    return 0;
}
```

## Advanced Driver Features in C/C++

### DMA Transfer Support

For high-throughput SPI, using DMA instead of PIO is essential. The SPI core provides helpers:

```c
/**
 * mysoc_transfer_one_dma - DMA-based transfer_one implementation
 *
 * Uses the kernel's dmaengine API to offload data movement to a DMA controller,
 * freeing the CPU from polling or interrupt-per-byte overhead.
 */
static int mysoc_transfer_one_dma(struct spi_controller *ctlr,
                                   struct spi_device *spi,
                                   struct spi_transfer *xfer)
{
    struct mysoc_spi *priv = spi_controller_get_devdata(ctlr);
    struct dma_async_tx_descriptor *tx_desc, *rx_desc;
    dma_cookie_t tx_cookie, rx_cookie;

    reinit_completion(&priv->completion);

    /* Map buffers for DMA (the SPI core does this if ctlr->can_dma returns true) */
    /* TX descriptor */
    tx_desc = dmaengine_prep_slave_single(ctlr->dma_tx, xfer->tx_dma,
                                          xfer->len, DMA_MEM_TO_DEV,
                                          DMA_PREP_INTERRUPT | DMA_CTRL_ACK);
    if (!tx_desc)
        return -ENOMEM;

    /* RX descriptor — callback fires on completion of RX (the slower channel) */
    rx_desc = dmaengine_prep_slave_single(ctlr->dma_rx, xfer->rx_dma,
                                          xfer->len, DMA_DEV_TO_MEM,
                                          DMA_PREP_INTERRUPT | DMA_CTRL_ACK);
    if (!rx_desc)
        return -ENOMEM;

    rx_desc->callback        = mysoc_dma_rx_callback;
    rx_desc->callback_param  = priv;

    /* Submit and fire */
    rx_cookie = dmaengine_submit(rx_desc);
    tx_cookie = dmaengine_submit(tx_desc);

    if (dma_submit_error(rx_cookie) || dma_submit_error(tx_cookie))
        return -EIO;

    dma_async_issue_pending(ctlr->dma_rx);
    dma_async_issue_pending(ctlr->dma_tx);

    /* Enable DMA request lines in the SPI hardware */
    mysoc_set_bits(priv, MYSOC_SPI_CTRL, CTRL_DMA_EN);

    /* Wait for RX DMA completion (RX always finishes last) */
    if (!wait_for_completion_timeout(&priv->completion, msecs_to_jiffies(1000))) {
        dmaengine_terminate_sync(ctlr->dma_tx);
        dmaengine_terminate_sync(ctlr->dma_rx);
        return -ETIMEDOUT;
    }

    mysoc_clr_bits(priv, MYSOC_SPI_CTRL, CTRL_DMA_EN);
    return 0;
}

static void mysoc_dma_rx_callback(void *data)
{
    struct mysoc_spi *priv = data;
    complete(&priv->completion);
}
```

### Power Management

```c
#ifdef CONFIG_PM_SLEEP
static int mysoc_spi_suspend(struct device *dev)
{
    struct spi_controller *ctlr = dev_get_drvdata(dev);
    struct mysoc_spi *priv = spi_controller_get_devdata(ctlr);
    int ret;

    ret = spi_controller_suspend(ctlr);
    if (ret)
        return ret;

    /* Save register state if hardware doesn't retain it across suspend */
    priv->saved_ctrl = mysoc_rd(priv, MYSOC_SPI_CTRL);

    /* Disable controller and clock */
    mysoc_wr(priv, MYSOC_SPI_CTRL, 0);
    clk_disable_unprepare(priv->clk);
    return 0;

static int mysoc_spi_resume(struct device *dev)
{
    struct spi_controller *ctlr = dev_get_drvdata(dev);
    struct mysoc_spi *priv = spi_controller_get_devdata(ctlr);
    int ret;

    ret = clk_prepare_enable(priv->clk);
    if (ret)
        return ret;

    /* Restore hardware state */
    mysoc_wr(priv, MYSOC_SPI_CTRL, priv->saved_ctrl);
    mysoc_wr(priv, MYSOC_SPI_INT_EN, INT_TRANSFER_DONE | INT_ERROR);

    return spi_controller_resume(ctlr);
}
```

### Transfer Optimization: `prepare_message` Hook

```c
/**
 * mysoc_prepare_message - Called before a new spi_message begins.
 * Configure the hardware once per message rather than per transfer,
 * to minimize overhead when a message contains many transfers.
 */
static int mysoc_prepare_message(struct spi_controller *ctlr,
                                  struct spi_message *msg)
{
    struct mysoc_spi *priv = spi_controller_get_devdata(ctlr);
    struct spi_device *spi = msg->spi;

    /* Apply device-level clock and mode settings once for the whole message */
    mysoc_set_clock(priv, spi->max_speed_hz);

    /* Flush FIFOs */
    mysoc_set_bits(priv, MYSOC_SPI_CTRL, CTRL_FLUSH_TX | CTRL_FLUSH_RX);
    mysoc_clr_bits(priv, MYSOC_SPI_CTRL, CTRL_FLUSH_TX | CTRL_FLUSH_RX);

    return 0;
}
```

```rust
// drivers/spi/spi_mysoc_rust.rs

// SPDX-License-Identifier: GPL-2.0

//! MySoC SPI Master Controller Driver (Rust implementation)
//!
//! Demonstrates the Rust kernel abstractions for:
//! - Platform device drivers
//! - MMIO register access via IoMem
//! - Interrupt handling
//! - SPI controller registration

use kernel::{
    bindings,
    clk::Clk,
    error::{to_result, Result},
    io_mem::IoMem,
    irq,
    module_platform_driver,
    of,
    platform,
    prelude::*,
    spi::{self, Controller, Device, Transfer},
    sync::{Arc, Completion, SpinLock},
};

/// Hardware register offsets
mod regs {
    pub const CTRL:     usize = 0x00;
    pub const STATUS:   usize = 0x04;
    pub const CLK_DIV:  usize = 0x08;
    pub const TX_DATA:  usize = 0x0C;
    pub const RX_DATA:  usize = 0x10;
    pub const CS:       usize = 0x14;
    pub const INT_EN:   usize = 0x18;
    pub const INT_STAT: usize = 0x1C;
}

/// Control register bit masks
mod ctrl_bits {
    pub const ENABLE:          u32 = 1 << 0;
    pub const MASTER:          u32 = 1 << 1;
    pub const CPOL:            u32 = 1 << 2;
    pub const CPHA:            u32 = 1 << 3;
    pub const LSB_FIRST:       u32 = 1 << 4;
    pub const WORD_SIZE_SHIFT: u32 = 8;
    pub const WORD_SIZE_MASK:  u32 = 0xF << WORD_SIZE_SHIFT;
}

/// Interrupt bit masks
mod int_bits {
    pub const TX_EMPTY:       u32 = 1 << 0;
    pub const RX_FULL:        u32 = 1 << 1;
    pub const TRANSFER_DONE:  u32 = 1 << 2;
    pub const ERROR:          u32 = 1 << 3;
}

const FIFO_DEPTH: usize = 16;
const MAX_CS: u16 = 4;

/// Active transfer state, protected by the spinlock in MysocSpiInner
struct TransferState {
    tx_buf:  Option<*const u8>,
    rx_buf:  Option<*mut u8>,
    tx_len:  usize,
    rx_len:  usize,
    error:   i32,
}

// Safety: TransferState only accessed under SpinLock
unsafe impl Send for TransferState {}

/// Inner (lock-protected) driver state
struct MysocSpiInner {
    xfer: TransferState,
}

/// The main driver data structure
struct MysocSpiData {
    /// Memory-mapped IO region for the SPI controller registers
    base:       IoMem<0x100>,
    /// The SPI functional clock
    clk:        Clk,
    /// Completion signal for interrupt-driven transfers
    done:       Completion,
    /// Lock-protected mutable state
    inner:      SpinLock<MysocSpiInner>,
}

impl MysocSpiData {
    /// Read a 32-bit register
    fn read_reg(&self, offset: usize) -> u32 {
        // Safety: all offsets are within the IoMem region size
        unsafe { self.base.readl(offset) }
    }

    /// Write a 32-bit register
    fn write_reg(&self, offset: usize, val: u32) {
        // Safety: all offsets are within the IoMem region size
        unsafe { self.base.writel(val, offset) }
    }

    fn set_bits(&self, offset: usize, bits: u32) {
        self.write_reg(offset, self.read_reg(offset) | bits);
    }

    fn clr_bits(&self, offset: usize, bits: u32) {
        self.write_reg(offset, self.read_reg(offset) & !bits);
    }

    /// Initialize the SPI hardware block
    fn hw_init(&self) -> Result {
        // Disable first
        self.write_reg(regs::CTRL, 0);

        // Enable as master with 8-bit words
        let ctrl = ctrl_bits::ENABLE
            | ctrl_bits::MASTER
            | ((8u32 - 1) << ctrl_bits::WORD_SIZE_SHIFT);

        self.write_reg(regs::CTRL, ctrl);
        self.write_reg(regs::INT_EN, int_bits::TRANSFER_DONE | int_bits::ERROR);
        Ok(())
    }

    /// Set the SPI clock divider
    fn set_clock(&self, speed_hz: u32) -> Result<u32> {
        let clk_rate = self.clk.get_rate() as u32;
        if clk_rate == 0 || speed_hz == 0 {
            return Err(EINVAL);
        }

        // Round up to even divider, minimum 2
        let mut divider = clk_rate.div_ceil(speed_hz);
        if divider < 2 { divider = 2; }
        divider = (divider + 1) & !1; // align to even

        self.write_reg(regs::CLK_DIV, divider);
        Ok(clk_rate / divider)
    }

    /// Push bytes into the hardware TX FIFO
    fn fill_tx_fifo(&self, inner: &mut MysocSpiInner) {
        let mut count = 0;
        while inner.xfer.tx_len > 0 && count < FIFO_DEPTH {
            let status = self.read_reg(regs::STATUS);
            if status & (1 << 0) != 0 { // TX_FULL
                break;
            }
            let byte = match inner.xfer.tx_buf {
                Some(ptr) => {
                    // Safety: caller guarantees validity for tx_len bytes
                    let b = unsafe { *ptr };
                    inner.xfer.tx_buf = Some(unsafe { ptr.add(1) });
                    b as u32
                }
                None => 0x00, // dummy byte for receive-only
            };
            self.write_reg(regs::TX_DATA, byte);
            inner.xfer.tx_len -= 1;
            count += 1;
        }
    }

    /// Drain the hardware RX FIFO into the receive buffer
    fn drain_rx_fifo(&self, inner: &mut MysocSpiInner) {
        while inner.xfer.rx_len > 0 {
            let status = self.read_reg(regs::STATUS);
            if status & (1 << 3) != 0 { // RX_EMPTY
                break;
            }
            let data = self.read_reg(regs::RX_DATA) as u8;
            if let Some(ptr) = inner.xfer.rx_buf {
                // Safety: caller guarantees validity for rx_len bytes
                unsafe { *ptr = data; }
                inner.xfer.rx_buf = Some(unsafe { ptr.add(1) });
            }
            inner.xfer.rx_len -= 1;
        }
    }
}

/// Interrupt handler for the SPI controller
struct MysocIrqHandler {
    data: Arc<MysocSpiData>,
}

impl irq::Handler for MysocIrqHandler {
    type Data = Arc<MysocSpiData>;

    fn handle_irq(data: &Arc<MysocSpiData>) -> irq::Return {
        let stat = data.read_reg(regs::INT_STAT);
        // Acknowledge all interrupts
        data.write_reg(regs::INT_STAT, stat);

        if stat & int_bits::ERROR != 0 {
            let mut inner = data.inner.lock();
            inner.xfer.error = -(bindings::EIO as i32);
            drop(inner);
            data.done.complete_all();
            return irq::Return::Handled;
        }

        if stat & int_bits::TRANSFER_DONE != 0 {
            let mut inner = data.inner.lock();
            data.drain_rx_fifo(&mut inner);

            if inner.xfer.tx_len > 0 {
                data.fill_tx_fifo(&mut inner);
            } else {
                drop(inner);
                data.done.complete_all();
            }
        }

        irq::Return::Handled
    }
}

/// The SPI controller operations implementation
struct MysocSpiOps;

impl spi::ControllerOps for MysocSpiOps {
    type Data = Arc<MysocSpiData>;

    /// Validate and configure a new SPI device
    fn setup(data: &Arc<MysocSpiData>, device: &Device) -> Result {
        let bpw = device.bits_per_word();
        if !(4..=16).contains(&bpw) {
            pr_err!("Unsupported word size: {bpw}\n");
            return Err(EINVAL);
        }
        if device.max_speed_hz() > 50_000_000 {
            pr_err!("Speed exceeds 50 MHz maximum\n");
            return Err(EINVAL);
        }

        let mut ctrl = data.read_reg(regs::CTRL);
        ctrl &= !(ctrl_bits::CPOL | ctrl_bits::CPHA | ctrl_bits::LSB_FIRST);

        let mode = device.mode();
        if mode & bindings::SPI_CPOL as u32 != 0 { ctrl |= ctrl_bits::CPOL; }
        if mode & bindings::SPI_CPHA as u32 != 0 { ctrl |= ctrl_bits::CPHA; }
        if mode & bindings::SPI_LSB_FIRST as u32 != 0 { ctrl |= ctrl_bits::LSB_FIRST; }

        ctrl &= !ctrl_bits::WORD_SIZE_MASK;
        ctrl |= ((bpw as u32 - 1) << ctrl_bits::WORD_SIZE_SHIFT);
        data.write_reg(regs::CTRL, ctrl);

        Ok(())
    }

    /// Assert or deassert a chip select line
    fn set_cs(data: &Arc<MysocSpiData>, device: &Device, enable: bool) {
        let cs = device.chip_select() as u32;
        let reg = data.read_reg(regs::CS);
        let new_reg = if enable { reg | (1 << cs) } else { reg & !(1 << cs) };
        data.write_reg(regs::CS, new_reg);
    }

    /// Execute one SPI transfer
    fn transfer_one(
        data: &Arc<MysocSpiData>,
        _device: &Device,
        xfer: &Transfer,
    ) -> Result {
        let speed = xfer.speed_hz();
        data.set_clock(speed)?;
        data.done.reinit();

        {
            let mut inner = data.inner.lock();
            inner.xfer = TransferState {
                tx_buf: xfer.tx_buf().map(|s| s.as_ptr()),
                rx_buf: xfer.rx_buf().map(|s| s.as_mut_ptr()),
                tx_len: xfer.len(),
                rx_len: xfer.len(),
                error:  0,
            };
            data.fill_tx_fifo(&mut inner);
        }

        // Wait up to 1 second for the ISR to signal completion
        let completed = data.done.wait_for_completion_timeout(
            kernel::time::msecs_to_jiffies(1000)
        );
        if !completed {
            return Err(ETIMEDOUT);
        }

        let inner = data.inner.lock();
        if inner.xfer.error != 0 {
            return Err(Error::from_kernel_errno(inner.xfer.error));
        }
        Ok(())
    }
}

/// Platform driver implementation
struct MysocSpiDriver;

impl platform::Driver for MysocSpiDriver {
    type Data = (Arc<MysocSpiData>, Controller);
    const OF_MATCH_TABLE: Option<&'static [of::DeviceId]> = Some(&[
        of::DeviceId::new(b"myvendor,mysoc-spi\0"),
    ]);

    fn probe(dev: &mut platform::Device, _id_info: Option<&Self::IdInfo>) -> Result<Self::Data> {
        // Map MMIO registers
        let base = dev.ioremap_resource(0)?;

        // Get and enable clock
        let clk = Clk::get(dev, Some(c_str!("spi_clk")))?;
        clk.prepare_enable()?;

        // Build driver data
        let priv_data = Arc::try_new(MysocSpiData {
            base,
            clk,
            done:  Completion::new(),
            inner: unsafe {
                SpinLock::new(MysocSpiInner {
                    xfer: TransferState {
                        tx_buf: None, rx_buf: None,
                        tx_len: 0, rx_len: 0, error: 0,
                    },
                })
            },
        })?;

        // Initialise hardware
        priv_data.hw_init()?;

        // Register IRQ
        let irq_num = dev.irq(0)?;
        irq::request_irq::<MysocIrqHandler>(irq_num, priv_data.clone(), 0, c_str!("spi-mysoc"))?;

        // Allocate and register SPI controller
        let ctlr = Controller::alloc::<MysocSpiOps>(dev, priv_data.clone())?;
        ctlr.set_num_chipselect(MAX_CS);
        ctlr.set_mode_bits(bindings::SPI_CPOL | bindings::SPI_CPHA | bindings::SPI_LSB_FIRST);
        ctlr.set_max_speed_hz(50_000_000);

        ctlr.register(dev)?;

        pr_info!("MySoC SPI master registered (Rust driver)\n");
        Ok((priv_data, ctlr))
    }

    fn remove(_dev: &mut platform::Device, data: &Self::Data) {
        let (priv_data, _ctlr) = data;
        // Disable hardware
        priv_data.write_reg(regs::CTRL, 0);
        priv_data.write_reg(regs::INT_EN, 0);
        priv_data.clk.disable_unprepare();
    }
}
```

### Comparison: C vs Rust Patterns

| Aspect | C | Rust |
|--------|----|------|
| Memory safety | Manual; relies on conventions | Enforced by borrow checker at compile time |
| Resource cleanup | `devm_*` helpers + explicit `remove()` | RAII via `Drop`; misuse caught at compile time |
| Null pointer | Checked manually (`IS_ERR`, `if (!ptr)`) | `Option<T>` — null case is impossible to ignore |
| Error propagation | Return codes, `goto err_label` | `?` operator with `Result<T, Error>` |
| Concurrency | `spinlock_t`, manual lock/unlock | `SpinLock<T>` — data inside lock, unlock via `Drop` |
| IRQ handler data lifetime | Manually guaranteed by design | Enforced via `Arc` reference counting |
| Register access | Raw `readl`/`writel` (unsafe everywhere) | Wrapped `IoMem` — unsafety explicit and localized |

```c
// tools/spi/mysoc_spi_test.c — user-space loopback test (MOSI connected to MISO)

#include <fcntl.h>
#include <stdio.h>
#include <string.h>
#include <sys/ioctl.h>
#include <linux/spi/spidev.h>
#include <unistd.h>
#include <stdint.h>

int main(void)
{
    int fd = open("/dev/spidev0.0", O_RDWR);
    if (fd < 0) { perror("open"); return 1; }

    /* Set SPI mode */
    uint8_t mode = SPI_MODE_0;
    ioctl(fd, SPI_IOC_WR_MODE, &mode);

    /* Set 8-bit words and 1 MHz clock */
    uint8_t  bits = 8;
    uint32_t speed = 1000000;
    ioctl(fd, SPI_IOC_WR_BITS_PER_WORD, &bits);
    ioctl(fd, SPI_IOC_WR_MAX_SPEED_HZ, &speed);

    uint8_t tx[] = { 0xDE, 0xAD, 0xBE, 0xEF, 0xCA, 0xFE };
    uint8_t rx[sizeof(tx)] = { 0 };

    struct spi_ioc_transfer tr = {
        .tx_buf        = (unsigned long)tx,
        .rx_buf        = (unsigned long)rx,
        .len           = sizeof(tx),
        .speed_hz      = speed,
        .bits_per_word = bits,
        .delay_usecs   = 0,
    };

    if (ioctl(fd, SPI_IOC_MESSAGE(1), &tr) < 0) {
        perror("SPI transfer"); close(fd); return 1;
    }

    printf("TX: ");
    for (size_t i = 0; i < sizeof(tx); i++) printf("%02X ", tx[i]);
    printf("\nRX: ");
    for (size_t i = 0; i < sizeof(rx); i++) printf("%02X ", rx[i]);
    printf("\n");

    /* In loopback mode, TX and RX should match */
    if (memcmp(tx, rx, sizeof(tx)) == 0)
        printf("LOOPBACK TEST PASSED\n");
    else
        printf("LOOPBACK TEST FAILED\n");

    close(fd);
    return 0;
}
```

## Summary

Implementing a **Linux SPI Master Controller Driver** involves writing a platform driver that bridges the gap between the generic SPI subsystem and the specific hardware registers of a given SoC. The key concepts and takeaways are:

**Architecture.** Linux's SPI stack is cleanly layered: user space talks to spidev or device-specific drivers; those call the SPI core via `spi_sync()`/`spi_async()`; the core schedules transfers onto the controller driver through the `transfer_one` or `transfer_one_message` callback.

**Core callbacks.** A minimal controller driver must implement `setup()` (validate and configure per-device parameters), `set_cs()` (chip select toggling), and `transfer_one()` (the actual data movement — PIO or DMA). The SPI core handles the rest: queuing, locking, and calling completion.

**PIO vs DMA.** For low-speed or short transfers, FIFO-based PIO driven by interrupts is sufficient. For sustained high-throughput (e.g., display controllers, fast flash), a DMA engine path (`dmaengine_prep_slave_single`) offloads memory movement from the CPU entirely.

**Resource management.** Modern drivers use `devm_*` APIs so that clocks, IRQs, and MMIO regions are automatically released when the device is unbound, reducing cleanup boilerplate and risk of resource leaks.

**Rust.** The Rust kernel abstractions offer compile-time memory safety, mandatory error handling via `Result`/`Option`, and RAII-based resource management — all of which address the most common bug categories in C kernel drivers. The patterns map closely to their C equivalents but with safety guarantees enforced by the compiler rather than code review.

**Device Tree.** Platform matching is done via the `compatible` string in the DT node. The driver's `of_match_table` causes the kernel to call `probe()` with the matched device, providing resource descriptions (reg, interrupts, clocks) without hardcoding addresses.

**Testing.** The `spidev` interface and `spidev_test` tool allow user-space loopback and functional tests without a custom slave device. Kernel dynamic debug and ftrace provide deep visibility into the transfer pipeline during driver bring-up.

Together, these elements form a robust, production-quality SPI Master Controller Driver suitable for upstreaming into the Linux kernel.
